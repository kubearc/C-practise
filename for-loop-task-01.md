##  Task: Print Multiplication Table

###  Objective

Write a C program that  multiplication table up to 10 using a `for` loop.

---

###  Requirements

* Take an integer number
* Use a `for` loop
* Print the table from 1 to 10

---

###  Sample Output

```
Enter a number: 5

5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
...
5 x 10 = 50
```
