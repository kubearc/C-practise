# Task 1: Grade Calculator

###  Problem:

Declare `int marks = 75;`

Use `if-else if` to print grade according to marks:

* 90 and above → `"Grade A"`
* 75 to 89 → `"Grade B"`
* 50 to 74 → `"Grade C"`
* Below 50 → `"Fail"`
